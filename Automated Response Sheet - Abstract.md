# ABSTRACT

**Title:** Automated Response Sheet

**Abstract:** 
The Automated Response Sheet system simplifies the evaluation process of objective answer sheets using image processing and Optical Mark Recognition (OMR). It scans answer sheets, detects marked responses, compares them with answer keys, and generates results automatically. This reduces manual effort, improves accuracy, and saves time in educational institutions.

**Frontend Tools:**
- HTML, CSS, JavaScript (for user interface), or simple GUI using Tkinter.

**Backend Tools:**
- Python, OpenCV, NumPy, Pandas, SQLite/MySQL for data storage.

**Hardware Requirements:**
- Computer/Laptop (4GB RAM)
- Scanner/Camera
- Storage Device

**Software Requirements:**
- Windows/Linux
- Python
- OpenCV, NumPy, Pandas
- VS Code/PyCharm

**Input:**
- Scanned OMR Sheets
- Answer Key

**Expected Output:**
- Student Marks
- Auto Evaluation Result
- Performance Report
